# Impulse Response Attribution

All impulse response files in this directory are derived from the
[Open Acoustic Impulse Response (OpenAIR) Library](https://www.openair.hosted.york.ac.uk/),
University of York, and are licensed under
[Creative Commons Attribution 4.0 International (CC BY 4.0)](https://creativecommons.org/licenses/by/4.0/).

Each file below is reproduced from the original OpenAIR distribution. `cave.wav`
and `hall.wav` are 4-channel B-format (FuMa: W, X, Y, Z); the others are mono or stereo.
B-format files are decoded to virtual Blumlein stereo at load time (see `src/fxTrack.js`).
